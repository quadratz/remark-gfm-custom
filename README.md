# remark-gfm-custom

A configurable remark-gfm that allows you to toggle the GitHub Flavored Markdown (GFM) features.

## Installation

```bash
bun add remark-gfm-custom
```

## Usage

```typescript
import { greet } from 'remark-gfm-custom';

console.log(greet('World')); // Hello, World!
```
